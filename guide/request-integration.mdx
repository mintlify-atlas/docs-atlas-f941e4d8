title: "Request an integration"
description: Learn how to request a new app integration for Automatisch.
---

If the app you need isn't listed in [available apps](/guide/available-apps), you can request it by opening a GitHub issue. The Automatisch team reviews all requests and prioritizes integrations based on community interest — so the more votes and comments an issue receives, the sooner it gets built.

Before opening a new issue, search the [existing issues](https://github.com/automatisch/automatisch/issues) to see if someone has already requested the same integration. If they have, upvote the issue and leave a comment describing your use case. This helps the team gauge demand.

<Tip>
  If an integration request already exists for the service you need, commenting on it is just as valuable as creating a new one. Community signals directly influence prioritization.
</Tip>

## Build it yourself

If you need an integration sooner, you can build it using the Automatisch integration SDK. The SDK is well-documented and designed so that any developer familiar with REST APIs can add a new app.

See the [build integrations guide](/build-integrations/overview) to get started.
